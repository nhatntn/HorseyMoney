# Nhạc nền khi đua

Đặt file nhạc nền (MP3) với tên **race-bg.mp3** vào thư mục này.

- Phát khi bắt đầu đua (sau countdown 3-2-1).
- Tự dừng khi kết thúc đua.

**Kích thước gợi ý:** khoảng **0,5–2 MB** (MP3 128 kbps, loop 1–2 phút) để tải nhanh trên mobile.

Bạn có thể dùng nhạc không bản quyền (royalty-free) hoặc nhạc tự tạo.
